# Instructions pour résoudre ChunkLoadError

Si vous rencontrez l'erreur `ChunkLoadError: Loading chunk app/layout failed`, suivez ces étapes :

1. **Arrêtez le serveur de développement** (Ctrl+C)

2. **Supprimez le cache Next.js** :
   ```powershell
   cd frontend
   Remove-Item -Recurse -Force .next -ErrorAction SilentlyContinue
   ```

3. **Nettoyez le cache npm** (optionnel) :
   ```powershell
   npm cache clean --force
   ```

4. **Redémarrez le serveur** :
   ```powershell
   npm run dev
   ```

5. **Videz le cache du navigateur** :
   - Chrome/Edge : Ctrl+Shift+Delete
   - Ou ouvrez en navigation privée

Si le problème persiste, essayez :
- Redémarrer complètement votre ordinateur
- Vérifier que le port 3000 n'est pas utilisé par un autre processus
- Mettre à jour Next.js : `npm install next@latest`
